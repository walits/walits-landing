Walits Custody API

거래소, 은행 및 기관 자산 관리자를 위한 엔터프라이즈급 MPC 2-of-3 커스터디 솔루션

Walits Custody란?

Walits Custody는 MPC 2-of-3 임계값 서명, 정책 기반 거버넌스, 자가 복구 메커니즘을 통해 기업 자산을 안전하게 보호하는 B2B 암호화폐 커스터디 솔루션입니다. Fireblocks/BitGo와 달리 Walits은 항상 키 공유의 1/3을 보유하여 단일 장애점(SPOF) 위험을 제거합니다.

주요 기능

MPC 2-of-3 보안

자가 복구 기능을 갖춘 CGGMP24 프로토콜 - 항상 1/3 키 공유를 제어합니다

옴니버스 + 가상 계정

자동 스윕 및 통합 커스터디를 통한 효율적인 자산 관리

정책 기반 거버넌스

N-of-M 승인 워크플로우, 지출 한도, 화이트리스트, 시간 잠금

실시간 웹훅

입금, 출금 및 승인에 대한 즉시 알림

지원 블록체인

Ethereum Mainnet Sepolia Testnet Base Polygon Bitcoin (출시 예정)
