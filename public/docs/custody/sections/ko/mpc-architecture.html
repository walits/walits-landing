MPC란?

다자간 연산(MPC)은 여러 당사자가 완전한 개인 키에 접근하지 않고도 암호화 서명을 공동으로 계산할 수 있게 합니다. 이는 단일 장애점을 제거합니다.

기술 스택

  • MPC 서버: Rust로 작성된 고성능 CGGMP24 구현
  • walits-provider: NestJS 기반 RPC Router 및 트랜잭션 추적
  • xScanner: Rust 기반 블록체인 모니터링 시스템

서명 프로세스

트랜잭션을 실행하려면 3개의 키 공유 중 2개가 서명 프로세스에 참여해야 합니다:

  1. 출금 요청: API 또는 대시보드를 통해 시작됨
  2. 정책 확인: 지갑 정책에 따라 승인 워크플로우 트리거됨
  3. 승인 완료: 필요한 승인 충족 (예: 2-of-3)
  4. MPC 서명: 귀사의 공유 + Walits 공유 → 서명 생성
  5. 브로드캐스트: 서명된 트랜잭션이 블록체인으로 전송됨

자가 복구 (SPOF 제거)

자가 복구 메커니즘

Walits은 귀사가 항상 키 공유의 1/3을 보유하도록 보장하여, 벤더 종속성을 제거합니다.

재해 시나리오: Walits 서비스 종료

  • 보유: 귀사의 키 공유 (1/3) + 백업 공유 (1/3) = 2/3
  • 가능: 전체 키 재구성 후 다른 지갑 솔루션으로 마이그레이션
  • 다운타임 제로: 귀사의 자금은 항상 접근 가능

기술 사양

프로토콜 CGGMP24 (2024 표준)
커브 secp256k1 (Ethereum, Bitcoin)
임계값 2-of-3 (t=2, n=3)
키 생성 시간 <30초
서명 시간 ~1.5초
처리량 100+ 서명/초 (병렬 처리)
구현 언어 Rust (MPC), NestJS (Provider)
