Walits Non-Custody API

💎 관리형 B2B 솔루션

게임 스튜디오, 커뮤니티 앱, 리워드 플랫폼에서 수천 명의 사용자 지갑을 가상 잔액 전송과 최소 가스 비용으로 관리할 수 있습니다.

Non-Custody란?

Walits Non-Custody는 다음과 같은 B2B 지갑 인프라입니다:

  • 플랫폼이 마스터 지갑과 모든 파생 서브 계정을 소유합니다
  • 최종 사용자는 HD 지갑 파생(BIP32/BIP44)을 통해 고유한 블록체인 주소를 받습니다
  • 가상 잔액을 통해 가스 비용 없이 즉시 P2P 전송이 가능합니다 (1,000+ TPS)
  • TEE 보안 (AWS Nitro Enclave)이 개인 키를 보호합니다

주요 기능

🔑 HD Wallet

BIP32/BIP44 표준을 사용하여 단일 니모닉에서 무제한 서브 계정을 자동 생성합니다

⚡ Virtual Balance

데이터베이스 레벨 잔액 업데이트를 통해 즉시 전송 (50-100ms), 1,000+ TPS, 가스 비용 무료

🔒 TEE Security

AWS Nitro Enclave에서 개인 키를 암호화하여 저장 (하드웨어 레벨 격리)

🪙 Custom Tokens

하나의 API 호출로 Ethereum/Base에 자체 ERC-20 토큰을 배포합니다
